# Job Insights - Project

### English:

#### What did I develop?

I implemented analyses based on a dataset about jobs.

The data was extracted from the [Glassdoor](https://www.glassdoor.com.br/) website and obtained through [Kaggle](https://www.kaggle.com/datasets/atharvap329/glassdoor-data-science-job-data), a platform providing datasets for data scientists.

#### Skills tested in the project:

- Using the Python interactive terminal ("Python shell" or "Python REPL" (Read-Eval-Print Loop)).
- Using conditional and loop structures.
- Using built-in Python functions.
- Handling exceptions.
- Performing file manipulation.
- Writing functions.
- Writing tests with Pytest.
- Writing your own modules and importing them into other code.
